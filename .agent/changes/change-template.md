# Change Template

Use this template for meaningful project changes.

```md
## YYYY-MM-DD — Change title

### Changed

- List concrete changes.

### Why

Explain the intent, feedback, or decision that caused the change.

### Related semantic items

- PAGE-000
- COMP-000
- DATA-000

### Related lessons

- LESSON-000

### Metrics affected

- METRIC-000

### Follow-up

- List open work or validation steps.
```
