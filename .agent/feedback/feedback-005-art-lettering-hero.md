# FEEDBACK-005 — Hero typography should feel like art lettering

Timestamp: 2026-06-24 00:18 ET  
Status: converted to active lesson  
Source: Homepage type treatment feedback

## Public-safe summary

The hero title feels too heavy, tight, centered, and poster-like. The hero should feel more like soft art lettering: printed or drawn, handmade, readable, and slightly playful.

## Extracted signal

Change the typography system, not just the font. Use softer display treatment, less extreme weight, looser line-height, gentler letter spacing, and a paper-list structure so the hero feels like an art-studio page.

## Affected semantic items

- PAGE-001 Homepage
- COMP-003 Scroll Title Sequence
- DESIGN-002 Large Editorial Type
- DESIGN-013 Art Lettering Typography
- DESIGN-014 Paper List Hero Lockup
- METRIC-013 Artsy Not Poster-Heavy

## Created lesson

- LESSON-014 Make hero typography feel like art lettering

## Implementation ideas

- use a softer display font stack
- reduce display weight from 900 to 700 or 800
- loosen letter spacing and line-height
- soften text color away from harsh near-black
- structure Welcome → To → Soft Strange Studio as a bullet-list/paper lockup
- keep scroll rotation but make the hero readable quickly
