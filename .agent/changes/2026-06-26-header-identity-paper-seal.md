# 2026-06-26 — Header identity paper seal

## Summary

The shared header identity mark now gets its own clipped paper/photo seal so `Angel Berger` reads as a placed studio label within the tactile top strip instead of plain text floating on the header surface.

## Interface change

- Added `app/header-identity-paper-seal-pass.css`.
- Loaded the pass from `app/layout.jsx` after the existing header paper edge stamps.
- The identity link now has a cut-paper label shape, stronger lifted-paper shadow, a small honest abstract paper/photo seal, and a retained visible focus outline.

## Why

Active header feedback and lessons ask the whole shared top bar to read as layered paper at screenshot scale. The navigation tabs already had several paper passes, but the identity side still remained comparatively text-first. This pass strengthens the identity half of the same header system without adding a `Pages` button, route numbers, helper text, fake content, or a competing navigation pattern.

## Satisfaction state

Partially satisfied: the header already had paper-strip and tab cues, but the identity label needed a clearer physical studio-mark treatment.
