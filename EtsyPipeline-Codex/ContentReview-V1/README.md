# ContentReview-V1

## Purpose

Review generated content for clarity, usefulness, and Etsy-readiness.

## Inputs

- Generated media from `MediaGeneration-V1`
- Raw media metadata when context is needed
- Review criteria for the current pass

## Outputs

- Review notes
- Approval or rejection signals
- Specific changes needed before the next pass

## Conventions

- Keep review feedback specific and actionable.
- Separate factual issues from preference-based changes.
- Preserve the reason an asset was accepted, revised, or rejected.
