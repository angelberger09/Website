# 2026-06-28 — Notes path direct exit slips

## Changed

- Added `app/notes-path-direct-exit-slips-pass.css`.
- Loaded it from `app/layout.jsx` immediately after `notes-path-exit-ledger-pass.css`.
- Replaced the Notes after-shelf path card numeric marker surface with direct room-slip labels: `read latest`, `archive room`, and `front door`.
- Added a calmer reader-path desk label, stitched path strip, and slightly stronger shared board/card shadows while keeping the same public Notes copy and route destinations.

## Why

The Notes `After the shelf` exit path was already a composed reader exit ledger, but its card marker still used visible numeric tokens from `data-path-index`. That partially conflicted with the active direct-navigation lesson and made the shelf read more like ordered UI cards than direct room choices. This pass keeps the reader path visible while making the exit options more human-facing and paper-like.

## Satisfaction state

Partially satisfied. This improves the Notes exit-path surface, but deployed review should still confirm that no other route/path cue reintroduces decorative numbering.
