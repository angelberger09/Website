# Vocabulary

## Studio pages

The visible page destinations on the homepage.

## Public feed

Machine-readable public JSON that the Website can safely read from another public repo.

## Semantic item

Any page, component, content source, interaction, design pattern, decision, lesson, or metric that carries project meaning.

## Lesson

A reusable rule distilled from feedback or a project decision.

## Matrix

A structured Markdown table that tracks semantic items, relationships, status, and metrics.

## Public memory

Distilled project context that is safe to store in the public repo.

## Authored

A design/copy quality where the site feels intentional, personal, and human-made rather than like a default test page.

## Technical demo language

Visible copy that sounds like a deployment test, raw endpoint list, or engineering note instead of a public-facing website.
