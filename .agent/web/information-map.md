# Information Map

This file maps what information exists, where it lives, and how the Website can use it.

## Website information

- Homepage structure lives in `app/page.jsx`.
- Global visual language lives in `app/globals.css`.
- Site metadata lives in `app/layout.jsx`.

## Blog information

- Blog Home is a public HTML page.
- Posts Index is a public HTML archive.
- Blog JSON Feed is the machine-readable list of published posts.
- Example Post is a full public entry.

## Agent information

- `agent.md` is the root editing guide.
- `start-here.md` is the human-friendly project intro.
- `.agent/memory.md` stores distilled public project memory.
- `.agent/matrices/*` stores semantic matrices.
- `.agent/lessons/*` stores reusable rules from feedback.

## Future information

- Portfolio feed will describe public projects.
- Store feed will describe public products or listings.
- Future notes viewer may render Blog post content inside Website.
