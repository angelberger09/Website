# Store direct paper cues

Timestamp: 2026-06-26 06:10 ET

## Interface mismatch

The Store page already had strong paper/photo sections, but several repeated Store surfaces still showed decorative numeric markers inside lane photos, gallery cards, checklist notes, and future-feed ledger tabs. That made the long Store path feel partly indexed like UI machinery instead of labeled as a calm room path.

## Satisfaction state

Partially satisfied.

## Visible Website change

Added `app/store-direct-paper-cues-pass.css` and loaded it from `app/layout.jsx`. The new pass replaces visible numeric markers on Store lane, gallery, checklist, and feed-ledger surfaces with direct clipped paper cues such as `Soft goods`, `Print shelf`, `Studio tools`, `Preview rule`, `Public path`, and `Delivery`.

## Why

Active direct-label and paper-snippet lessons support using readable paper labels instead of decorative numbering when labels can clarify the visitor path. The change keeps all Store content honest and does not imply live inventory, fake product photos, or unavailable listings.
