# MediaGeneration-V1

## Purpose

Generate or prepare media assets for Etsy-oriented workflows.

## Inputs

- Raw media inventory entries.
- The paired raw asset and `manifest.json`.
- Any generation brief or transformation goal.

## Outputs

- Generated media candidates.
- Notes describing what changed and why.
- Any constraints that should carry into review.

## Conventions

- Keep generation decisions explicit.
- Preserve the asset/manifest pair model when new media is added.
- Prefer small modular changes that can be overridden later.
