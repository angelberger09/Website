# Footer exit receipt photo cues

Timestamp: 2026-06-26 14:18 ET

## Interface change

The shared footer exit receipt shelf now includes honest abstract paper/photo cue faces for the Start, Read, and Browse receipts. `app/site-chrome.jsx` adds a visual label for each receipt, and `app/footer-exit-receipts-pass.css` lays each receipt out as a compact photo-led paper card with the original copy kept readable beside it.

## Why

The footer exit shelf was already a useful guided path, but it remained mostly text-first compared with the newer route-photo and paper-snippet patterns. Current `.agent` guidance asks shared route and continuity surfaces to carry meaningful photo/paper centers when safe, without inventing fake screenshots, product photos, portfolio images, or private material.

## Satisfaction state

Partially satisfied: the footer exit shelf is now visibly closer to the paper/photo system, while future passes should avoid repeatedly touching the same footer area unless a new mismatch appears.
