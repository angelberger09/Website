## Goal

Make the homepage current-room status area read as one calm, guided room-status compass instead of four detached paper cards.

## Boundary

- Keep existing homepage room data and visitor-facing copy.
- Do not invent products, posts, portfolio records, photos, or private content.
- Preserve links, route behavior, and accessible source text.
- Use CSS-only material grouping so the change remains low-risk and visible.

## Quality metric

The section should visibly show:

- a left-docked room-status label,
- a connecting stitched thread,
- lifted paper/card depth,
- direct readiness cue stamps,
- mobile fallback that stacks without hiding content.
