# .agent Project Memory

This folder is a public project-memory system for the Website repo.

It stores:

- project intent
- distilled feedback lessons
- semantic item matrices
- design and copy rules
- public data-source maps
- quality metrics
- change history
- reusable templates

## Public safety rule

Only store public-safe distilled lessons here. Do not store raw private conversations, vulnerable personal details, finances, health details, or private drafts.

## Main workflow

1. Read `../agent.md` and `.agent/start-here.md` before editing.
2. Use `.agent/matrices/semantic-index.md` to understand meaningful site items.
3. Apply active lessons from `.agent/lessons/active-lessons.md`.
4. After changes, update the change log, matrices, memory, and relationship maps as needed.
