# 2026-06-25 — About room-map photo centers

## Changed

- Added `aboutRoomFrames` to `app/about/page.jsx` so each opening About room-map card has a public-safe abstract visual center.
- Added `app/about-room-photo-pass.css` so those centers render as clipped paper/photo pieces with tape, grain, scratch, label, and mobile fallbacks.
- Imported the new pass from `app/layout.jsx`.
- Updated page and metric matrices to record the visible About refinement.

## Why

The About opening room map had already moved away from a generic card grid, but the first orientation cards still read more text-first than the newer photo-led surfaces on About, Notes, Portfolio, Store, and the homepage. This pass keeps the existing public-safe About copy and adds honest abstract paper/photo centers that do not imply private photos, fake portfolio work, or unavailable assets.
