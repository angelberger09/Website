# Project Intent

## Purpose

The Website repo is the public front door for Soft Strange Studio.

It should introduce Angel Berger’s studio presence, guide people toward public notes and future work, and eventually connect cleanly to Blog, Portfolio, and Store data.

## Audience

- visitors looking for Angel Berger / Soft Strange Studio
- future readers of public notes
- future clients or collaborators
- future assistants or developers working in the repo

## Tone

Minimal, warm, authored, gentle, and studio-like.

## What this site is not

- not a raw deployment test page
- not a technical dashboard
- not a place for private drafts
- not a full data portal yet

## Long-term direction

A small public studio system where the main Website presents the front door, while Blog, Portfolio, and Store repos can publish structured public content feeds.
