# 2026-06-26 — Homepage path ledger focus

## Changed

- Added `app/home-path-ledger-focus-pass.css`.
- Loaded the pass from `app/layout.jsx` after the existing homepage and shared paper passes.
- Refined the existing homepage `home-path-map` reading path into one left-labeled paper ledger shelf with native horizontal overflow on wider screens and a single-column fallback on small screens.

## Why

The homepage reading path already made the long page more legible, but its six equal cards could still read like another card grid. Current active lessons ask the homepage to behave as a human-readable left-docked editorial path and ask major pages to feel like calm rooms rather than accumulations of loose scraps. This pass keeps the same public-safe anchors and copy while making the orientation map feel like one composed path object.
