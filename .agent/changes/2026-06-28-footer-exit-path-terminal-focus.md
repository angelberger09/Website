## In-bounds mismatch

The shared footer already had a terminal room ledger pass, current-room stamp, exit receipts, and direct footer navigation. The remaining mismatch was that the receipt shelf could still scan as adjacent lower-page scraps instead of one calm guided exit path.

The pass:

- keeps all existing public-safe footer copy and links,
- strengthens receipt photo centers and action slips,
- gives the lower footer a clearer two-column-to-one-column responsive rhythm,
- preserves accessible links and reduced-motion behavior.
