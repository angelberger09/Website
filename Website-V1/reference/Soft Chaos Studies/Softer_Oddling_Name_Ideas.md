# Softer Oddling Name Ideas

## Anxiety / nervous system friends

- Watchful Friend
  - Soft, alert, protective. This is probably the gentlest anxiety-coded name.
- Trying-My-Best Friend
  - Funny, relatable, a little messy, very sticker-friendly.
- Overthinking Oddling
  - More direct, but still soft and cute.
- Tiny Spiral Friend
  - Great for anxiety, rumination, dissociation, or "my brain is doing loops."
- Soft Alarm Friend
  - A creature that is trying to protect you, even when it is overreacting.
- Pocket Panic Pal
  - More playful and funny, probably good for stickers.

## Dino / keeper friends

- Dino Keeper
  - Still the favorite. Clear, cute, and flexible.
- Fern Friend
  - Gentler and more nature-based.
- Mossy Keeper
  - Feels cottagecore and prehistoric at the same time.
- Little Fossil Friend
  - Cute for ancient feelings, memory, grief, and dinosaurs.
- Pebbleback Pal
  - A tiny stegosaurus / triceratops type friend with a soft, collectible feel.

## Garden / nature friends

- Bloom Buddy
  - Very stickerable and cheerful.
- Moss Pocket
  - This feels like a creature and a tiny world.
- Garden Keeper
  - Simple, cozy, and very brand-safe.
- Sproutling
  - Tiny, hopeful, emotionally soft.
- Tucked-In Tulip
  - Sweet, sleepy, and comfort-object coded.

## Ocean friends

- Shell Keeper
  - Absolutely keep this one. It feels gentle, collectible, and a little mysterious.
- Tidepool Friend
  - Soft, curious, good for ocean-themed coloring pages.
- Little Drift Friend
  - For going with the flow, dissociation, grief, or emotional gentleness.
- Moonpool Oddling
  - A little more magical and dreamy.
- Sea Glass Friend
  - Pretty, healing-coded, and great for the pastel-earthy / rainbow palette.
