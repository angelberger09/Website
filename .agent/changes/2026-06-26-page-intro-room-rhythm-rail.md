# 2026-06-26 — PageIntro room rhythm rail

## Changed

- Added a shared `page-hero__room-rhythm` rail to `PageIntro` in `app/site-chrome.jsx`.
- Added `app/page-intro-room-rhythm-pass.css` so each major route intro now shows four clipped paper steps: Enter, Focus, Shelf, and Continue.
- Imported the pass from `app/layout.jsx` after the calm room and heading ribbon passes so it visibly supports the newer room-structure direction.

## Why

Active lessons now ask every major page to read as a calm room with a clear composed hero, focused content, secondary shelf, readiness/source note, and next-path rhythm. Shared page intros already carried room tags and a route photo card, but the actual reading rhythm was not yet visible at the top of each route. This pass makes that structure visible without adding fake content, fake assets, private material, or a new navigation system.

## Satisfaction

Partially satisfied. The shared route intro now visibly marks the room rhythm across About, Notes, Blog compatibility, Portfolio, and Store. Deployed visual review should confirm the rail stays readable and calm at mobile sizes.
