# Calm room page structure

This document describes the layout reset in `app/calm-structured-rooms-pass.css`.

## Shared rhythm

Major pages should use:

1. Header
2. Quiet page hero
3. Featured content
4. Secondary shelf or grid
5. Source, process, or readiness note
6. Next-room path
7. Footer

## Homepage

Use a smaller hero lockup, one composed paper preview, a clear Start Here path, and ordered sections.

## Notes

Use a reading shelf. The latest note receives the strongest position. Older notes sit in a calmer grid. Source and state details sit below the reading area.

## About

Use a studio map: short introduction, public studio shape, small principle notes, and simple next paths.

## Portfolio

Use an archive table: featured work, project shelf, process lane, and clear status labels.

## Store

Use a future shop shelf: prepared lanes, availability states, and clear launch guidance.

## Motion

Use opacity and transform only. Keep motion short. Support `prefers-reduced-motion`.
