## Implementation summary

The pass strengthens the homepage studio room doors as a composed paper-room ledger. It does not change visible copy or route destinations. It adds stronger paper backing, receipt labels, tape marks, lifted shadows, cue snippets, and image-center framing so the room doors feel more physically placed and easier to scan.
