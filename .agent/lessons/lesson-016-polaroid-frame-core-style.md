# LESSON-016 — Use polaroid frame stacks as a core material motif

Timestamp: 2026-06-24 00:28 ET  
Context: Generated polaroid frame and dark backing assets  
Status: active

## Lesson

The polaroid/photo-frame material should become part of the site's core visual language, not a one-off decoration.

## Future rule

When the site needs tactile depth, layer a darker torn-paper backing beneath a cream torn polaroid/photo frame. Keep it flat, alpha-ready, slightly imperfect, and integrated with the existing grain/gloss/scratch atmosphere.

## Metric

The frame stack should feel like a reusable Soft Strange Studio material system: tactile, handmade, slightly darker behind, and readable when layered with content.
