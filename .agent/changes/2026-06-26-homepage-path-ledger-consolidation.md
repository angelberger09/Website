# Homepage path ledger consolidation

Timestamp: 2026-06-26 17:30 ET

## Changed

- Added `app/home-path-ledger-consolidation-pass.css`.
- Loaded it after `app/home-path-map-room-cards-pass.css` in `app/layout.jsx`.
- Rebalanced the homepage `Read this page` path map into a quieter left-docked editorial ledger with smaller chapter slips, a direct `front-page map` stamp, tighter path spacing, and calmer mobile behavior.

## Why

The homepage reading path had already gained strong paper material and six connected room cards, but the later room-card styling risked undoing the earlier ledger-focus goal by making the orientation object feel like another dense card board. Current `.agent` state asks the homepage to stay human-readable, left-docked, editorial, and calm while preserving visible paper/photo material. This pass keeps the same anchors, copy, and public route data while making the visible path guide scan more like one composed front-page map.

## Satisfaction state

Partially satisfied. The homepage reading path is closer to the intended left-docked editorial guide, but deployed visual review should still confirm that the full homepage scroll rhythm is calmer at screenshot scale.
