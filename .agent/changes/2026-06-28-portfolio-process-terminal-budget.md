# 2026-06-28 — Portfolio process terminal budget

## Summary

Added a final Portfolio publishing-rhythm budget pass so the `Publishing rhythm` section reads as one calm paper room surface instead of a heavier cluster of late process/runway slips.

## Interface change

- Added `app/portfolio-process-terminal-budget-pass.css`.
- Loaded it from `app/layout.jsx` immediately after the existing Portfolio process/state docket pass.
- Softened the Portfolio process board backing, heading sheet, runway strip, step slips, photo centers, thread opacity, shadows, mobile layout, and reduced-motion behavior.

## Why

The Portfolio page already had visible process photo cards, public thread cues, runway styling, and docket labels. Because those late layers did not have a final process-specific budget pass, the publishing-rhythm section could still feel heavier and more separate than the surrounding Portfolio room. This pass keeps the same public-safe publishing copy and process labels while making the area read as a quieter bound paper receipt.
