# 2026-06-28 — About contact sheet direct cues

## Changed

- Updated `app/about-contact-sheet-frame-sequence-pass.css`.
- Replaced generated decorative `frame 01` / `02` style labels with direct named contact-sheet cues: `front room`, `tone cue`, `soft edge`, and `gentle motion`.
- Replaced the small tag-number suffixes with direct word cues: `front`, `tone`, `edge`, and `motion`.

## Why

The About contact sheet had become a visible frame path, but the latest styling reintroduced decorative numbering even though active navigation/path lessons prefer direct, unnumbered labels. This keeps the contact-sheet structure and public-safe About copy while making the cue language more human-readable and aligned with the direct-label rule.

## Satisfaction state

Partially satisfied. The contact sheet now avoids generated numeric frame labels, but deployed visual review should still confirm no other route/path surfaces use decorative numbers where direct room labels would be clearer.
