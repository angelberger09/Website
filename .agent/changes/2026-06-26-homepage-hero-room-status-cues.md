# 2026-06-26 — Homepage hero room status cues

## What changed

- Updated `app/home-entry-contact-sheet-pass.css` so the homepage hero contact-sheet cards no longer repeat the same visible `Open room` cue on every destination.
- The four first-screen route cards now show direct room states at scan scale: `Open map`, `Reading shelf`, `Public archive`, and `Preparing`.

## Why

The homepage hero already gave visitors four strong public-room doors, but the repeated status slip flattened every room into the same state. That was only partially aligned with the active honesty/readiness lessons because Store is still preparing and each room has a different job.

This pass keeps the existing route data, links, copy, and accessibility structure intact while making the visible first-screen contact sheet more honest and more useful.
