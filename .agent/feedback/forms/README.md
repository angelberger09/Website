# Feedback Forms

This folder stores split feedback forms when one screenshot or note contains several distinct implementation signals.

Use forms when a single feedback moment needs to become multiple bounded instructions instead of one oversized lesson.

## Current forms

- `2026-06-24-polaroid-not-decal.feedback.md` — polaroid/photo surfaces should not sit behind content as empty decals.
- `2026-06-24-placeholder-photo-centers.feedback.md` — use rendered placeholder photo/shot centers so the page looks professionally composed before live assets exist.
- `2026-06-24-alternating-scrollable-gallery.feedback.md` — create a scrollable gallery pattern with alternating photo/card rhythm.
- `2026-06-24-professional-shot-backgrounds.feedback.md` — photo areas need intentional shot backgrounds and content staging, not blank gray middles.

## Boundary

These are feedback-only records. They should guide a later implementation run and should not be treated as approval to edit public Website source in the same pass unless Angel explicitly asks for implementation.
