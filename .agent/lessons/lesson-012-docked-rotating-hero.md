# LESSON-012 — Dock hero type and reduce scroll length

Timestamp: 2026-06-24 00:05 ET  
Context: Homepage hero motion feedback  
Status: active

## Lesson

The Welcome → To → Soft Strange Studio hero should read as one left-docked title lockup, not three separate full-screen panels.

## Future rule

Use compact, sticky, left-aligned hero typography with subtle scroll-linked rotation when the goal is arrival energy without excess scrolling.

## Applies to

- PAGE-001 Homepage
- COMP-003 Scroll Title Sequence
- DESIGN-002 Large Editorial Type
- DESIGN-009 Animated Background Texture
- INT-009 Scroll-Rotating Hero Lockup

## Metric

The user should understand the full studio title quickly, with less vertical scroll and visible motion that feels intentional.
