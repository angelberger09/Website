# Agent Guide

This repo is the public Soft Strange Studio website for Angel Berger.

Before making design, content, data, or architecture changes, read:

1. `start-here.md`
2. `.agent/start-here.md`
3. `.agent/memory.md`
4. `.agent/matrices/semantic-index.md`
5. `.agent/lessons/active-lessons.md`

## Purpose

Use the `.agent/` folder as public project memory. It stores distilled, public-safe intent, feedback lessons, semantic item matrices, quality checks, and relationship maps.

## Rules for future edits

- Keep public-facing copy human, authored, and studio-like.
- Avoid visible technical demo language unless the page is explicitly for technical documentation.
- Update `.agent/changes/change-log.md` after meaningful changes.
- Update semantic matrices when pages, components, data sources, interactions, or design patterns change.
- Store only public-safe distilled lessons. Do not store raw private conversation details.

## Current north star

A minimal floating-header studio homepage for Angel Berger, with a quiet scroll-reveal welcome and a small folder menu that opens the site’s current public pages.
