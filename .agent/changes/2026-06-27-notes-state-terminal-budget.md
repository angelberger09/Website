# Notes state terminal budget

Date: 2026-06-27 21:21 ET

## Interface mismatch

The Notes reader behavior board had accumulated tray, direct-cue, and reader-docket styling, but the newest docket surface was not yet covered by a terminal paper budget. It was partially satisfying the calm-room goal: visibly composed, but still capable of reading heavier than nearby Notes room paper surfaces.

## Satisfaction state

Partially satisfied.

## Visible Website change

Added and loaded `app/notes-state-terminal-budget-pass.css` to keep the `.notes-state-board` as a visible `state ledger` while calming its:

- shared paper backing,
- vertical state thread,
- heading slip,
- individual state card shadows,
- photo-center depth,
- mobile transforms,
- reduced-motion behavior.

## Why this is in bounds

The change is limited to the Website repo, touches only the public Notes interface and public project ledger, preserves existing public-safe copy, does not invent notes or private material, and follows active paper/collage, calm-room, text-snippet, photo-surface, and terminal-budget lessons.
