# 2026-06-25 — Homepage direct path labels

## Changed

- Replaced homepage path-map markers from numeric labels to direct paper cues: `Enter`, `Ready`, `Doors`, `Shape`, `Rules`, and `Next`.
- Replaced homepage section docket text such as `01 / Front room` with direct section names.
- Added `app/home-direct-path-labels-pass.css` so the direct cues render as clipped paper labels while preserving the existing anchor-link reading path.
- Imported the pass in `app/layout.jsx` after the homepage docket styling.
- Updated the pages matrix for PAGE-001.

## Why

The homepage already had a visible editorial path, but the remaining decorative numbering conflicted with the direct-label navigation lesson and METRIC-014. The visible path should keep its guided paper structure while using human-readable room cues instead of numeric markers.
