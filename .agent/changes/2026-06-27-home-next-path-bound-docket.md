# 2026-06-27 — Homepage next path bound docket

## Interface change

- Added `app/home-next-path-bound-docket-pass.css`.
- Loaded it after the existing homepage next-path room/ticket-stack passes in `app/layout.jsx`.
- Strengthened the homepage closing `Next path` note into a bound exit docket with a visible `exit docket` slip, stronger lifted paper shadow, stitched connection into the shared continuity trail, and clipped continuation slips for the About/Notes path.

## Why

The homepage closing path already had a paper note and abstract `next room` face, but it could still read as one standalone card sitting before the shared continuity trail. This pass makes the closing surface feel more like a bound room-exit docket, improving the calm-room sequence, paper-first material, flattened hierarchy, text-snippet guidance, and physical continuity without changing routes, inventing content, adding fake imagery, or exposing private material.

## Satisfaction state
