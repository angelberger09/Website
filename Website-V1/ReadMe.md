# Website V1

Static multi-page Vite site for Angel Berger.

## Routes

- `/` - home page with page previews and contact path.
- `/physical/` - physical work, featured item, item list, material notes, inquiry links.
- `/digital/` - digital work, featured item, item list, screen-work notes, inquiry links.
- `/journal/` - featured note, visual rail, archive notes, inquiry links.
- `/contact/` - direct email, inquiry options, and static mailto form.

## Source Map

- `src/data/content.js` is the source of truth for site identity, nav, social links, page content, contact copy, and derived featured/list exports.
- `src/data/generated-images.js` contains generated placeholder image data. Replace this boundary when real images are ready.
- `src/components/` contains small vanilla JS render and behavior helpers for chrome sync, entries, modals, nav state, page previews, parallax, and visual rails.
- `src/styles/` contains tokens, base styles, layout, shared components, and page-specific styles.
- `dist/` is generated by Vite. Do not edit it by hand.

## Commands

```sh
npm run dev
npm run build
npm run preview
```
