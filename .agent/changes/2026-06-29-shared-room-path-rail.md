# Shared room path rail

Timestamp: 2026-06-29 01:11 ET

## What changed

Added and loaded `app/shared-room-path-rail-pass.css` for the shared `.site-shell.page-layout` surface used by the major Website rooms.

The pass adds:

- a subtle fixed left-edge paper rail on desktop,
- a vertical `public room path` label,
- grain and scratch texture from the existing atmosphere assets,
- a clipped handmade-paper silhouette,
- soft contact shadow depth,
- a quieter absolute rail fallback on smaller screens.

## Why

Current project state asks every major page to feel like a calm room with a composed path rather than separate stacked boards. Many route-specific rails already exist, but the shared major-page shell still needed one consistent connective room-thread layer so About, Notes, Portfolio, Store, and compatibility routes feel like parts of the same public studio path.

## Satisfaction state

Partially satisfied → refined.

## Public-safety notes

No fake posts, projects, products, screenshots, private content, or external data were added. The pass only adds a shared decorative/organizational paper rail using existing public page structure and existing texture assets.
