# Paper Header Surface Reference

Timestamp: 2026-06-24 19:41 ET

## Source

Screenshot feedback from the live `/Website/about/` page showing the shared top navigation/header.

## Public-safe visual note

The top bar should feel less like a smooth translucent UI pill and more like a tactile paper strip or paper tab sitting over the top of the page.

## Direction to preserve

- Keep the direct nav labels: About, Notes, Portfolio, Store.
- Keep the header attached to the top edge and readable.
- Use existing atmosphere or polaroid material assets where helpful, especially grain, gloss, scratches, or torn-paper alpha assets.
- A shader-like CSS treatment is also acceptable if it creates a paper surface effect without adding heavy realism.
- Add a visible but soft drop shadow so the bar feels physically layered over the page.
