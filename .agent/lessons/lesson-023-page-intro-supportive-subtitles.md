# LESSON-023 — Keep page intro titles short, thin, and supportive

Timestamp: 2026-06-24 19:43 ET  
Context: Screenshot review of current About page intro typography  
Status: active

## Lesson

Page intro titles across the public Website should not behave like oversized hero headlines. They should act as supportive subtitles that quietly orient the visitor.

## Future rule

Use 2–5 word page intro titles, keep them thinner and slightly smaller than the current heavy treatment, and constrain them so they remain on one line. Reserve true hero-scale display type for the homepage arrival moment or intentional feature sections only.

## Metric

METRIC-022 — One-Line Supportive Subtitle
