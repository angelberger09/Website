# 2026-06-27 — Store gallery slow shelf

## Interface change

- Added `app/store-gallery-slow-shelf-pass.css`.
- Loaded it after `app/late-paper-surface-budget-pass.css` in `app/layout.jsx` so the Store gallery refinement remains visible after the late shared budget pass.
- Turned the existing Store scrollable preview gallery into a more guided `slow shelf` sequence with a clipped shelf label, public preview slip, stitched guide thread, per-card preview/rule slips, and small paper tape anchors.

## Why

The Store gallery already used existing public-safe lane/rule data and alternating paper/photo cards, but it could still scan as a loose row of cards rather than one guided preview shelf. This pass makes the gallery more human-readable, paper-first, and aligned with the calm-room / alternating scroll-gallery metrics without changing Store copy, route links, data, accessibility labels, availability state, or inventing product imagery.
