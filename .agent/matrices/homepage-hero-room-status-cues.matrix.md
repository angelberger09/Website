## Interface mismatch

The homepage hero contact sheet had route cards with useful titles and descriptions, but every visible status slip repeated `Open room`. That made the first-screen route strip less honest and less informative, especially because the Store route is still a preparing path rather than a live shop.

## Visible correction

The route-card status slips now display direct room states:

- About: `Open map`
- Notes: `Reading shelf`
- Portfolio: `Public archive`
- Store: `Preparing`

## Boundary

This is a visible interface refinement only. It does not create fake products, fake listings, fake project records, fake screenshots, or private content. It preserves the existing route links and homepage card data while making the visible first-screen cues better aligned with the public-room and honest-readiness rules.
