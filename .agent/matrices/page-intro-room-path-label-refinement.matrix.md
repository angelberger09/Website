## Change relationship

The PageIntro shelf was already composed as a paper-room index. This pass keeps the same paper structure but changes its visible language and marks so it supports visitor orientation instead of internal indexing.

## Before

- Shelf label: `room index`.
- Each slip had visible numeric marks `01` through `04`.
- The result could read as a system index rather than a public room path.

## After

- Shelf label: `room path` on desktop and `path` on narrow mobile.
- Numeric marks became small paper-tape receipt marks.
- The shared shelf retains tactile paper detail while avoiding numbered/internal-feeling UI metadata.

## Validation target

Rendered route intros should still show four readable shelf slips, but the shared intro should feel warmer, more public-facing, and less system-indexed.
